## Tulang
A programming language created for special
course dedicated to development
programming language from 0


Fibonacci in tulang:
```rust
fn fibonacci(n) {
  if(n < 2) {
    return n
  }
  return fibonacci(n - 1) + fibonacci(n - 2)
}

println(fibonacci(10))

```
